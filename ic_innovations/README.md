# IC Innovations Tech & Business Management Consultancy

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![PHP Version](https://img.shields.io/badge/PHP-7.4%2B-blue)](https://php.net)

A complete responsive dynamic website for business management and tech consultancy with quotation system, admin panel, and service tracking.

## Features

- ✅ Fixed header with navigation (We, For You, Achieved, Communicate, I-Corner)
- ✅ Push sidebar with dynamic content
- ✅ Service quotation system with auto-calculation (tax, total)
- ✅ Admin panel to manage service charges & news
- ✅ User registration/login with auto-generated password
- ✅ Gantt-style project completion tracker
- ✅ Moving news bulletin (admin editable)
- ✅ Responsive design for all devices
- ✅ Security headers & CSRF protection
- ✅ PDO prepared statements (SQL injection prevention)

## Tech Stack

- **Backend:** PHP 7.4+ (PDO for database)
- **Database:** MySQL 5.7+
- **Frontend:** HTML5/CSS3/JavaScript (Vanilla)
- **Security:** bcrypt passwords, CSRF tokens, security headers

## Installation

### Prerequisites

- XAMPP/WAMP/LAMP stack
- PHP 7.4 or higher
- MySQL 5.7 or higher
